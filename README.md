# TinderBE

## Overview
Backend server for a Tinder-style matchmaking platform built using Node.js and Express. 
Handles user authentication, profile management, swipe logic, and match creation.

## Key Features
- User Registration & Login
- JWT-based Authentication
- Profile CRUD Operations
- Swipe & Match Logic
- Protected Routes

## Tech Stack
Backend: Node.js, Express  
Database: MongoDB (Mongoose ODM)  
Authentication: JWT  

## Run Locally
1. Install dependencies → `npm install`
2. Start server → `npm start`
3. Connect with TinderFE frontend
